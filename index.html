HTML

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Ele define o significado e a estrutura do conteúdo em páginas da internet.

Outras tecnologias, como CSS e JavaScript, são usadas em conjunto para definir a aparência e a interatividade das páginas.

CSS

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem usada para definir a apresentação de documentos HTML ou XML.

Ele permite controlar cores, fontes, espaçamento, layout e adaptação para diferentes dispositivos.

JavaScript

JavaScript é uma linguagem de programação leve, interpretada e orientada a objetos, amplamente usada para criar interações dinâmicas em páginas da web.

Além dos navegadores, também é utilizada em ambientes como Node.js e ferramentas de automação. Suporta múltiplos paradigmas de programação, como funcional e orientada a objetos.
